# Minhoquinha

Jogo clássico da minhoquinha desenvolvido com HTML, CSS e JavaScript puro.

## Como jogar

1. Abra o arquivo `index.html` em um navegador.
2. Use as setas do teclado para movimentar a minhoquinha.
3. Coma a fruta para aumentar a pontuação e crescer.
4. O jogo reinicia quando a minhoquinha colide com o próprio corpo.

Nenhuma instalação é necessária.
